# IoT Temperature Monitoring System

This project is built using ESP32 and DHT22 sensor in Wokwi.

## Features
- Temperature monitoring
- LED alert system
- Buzzer alert when temperature exceeds threshold

## Components
- ESP32
- DHT22 Sensor
- LED
- Buzzer

## Simulation
Built using Wokwi simulator

## How it works
- Reads temperature from DHT22
- If temperature > 30°C:
  - LED turns ON
  - Buzzer turns ON
